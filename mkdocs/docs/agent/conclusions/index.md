# Conclusions i Futures Millores

## Futures Millores

Com a línies de futur per a *Cybershield Solutions*, es proposa:

### Curt Termini

1. **Autenticació d'Usuaris**: Afegir sistema de login per protegir l'accés al tauler de control
2. **Alertes en Temps Real**: Implementar notificacions per correu electrònic o Telegram quan es detecti un atac de força bruta

### Mitjà Termini

3. **Base de Dades Persistent**: Crear una base de dades SQLite per guardar l'històric d'auditories i comparar l'evolució de la seguretat en el temps
4. **API Pública**: Documentar i exposar una API REST completa per integració amb altres eines

### Llarg Termini

5. **Suport Multi-Plataforma**: Ampliar compatibilitat a altres distribucions Linux (CentOS, Debian)
6. **Mode Clúster**: Permetre monitorar múltiples servidors des d'una única consola
7. **Machine Learning**: Incorporar detecció d'anomalies basada en IA
